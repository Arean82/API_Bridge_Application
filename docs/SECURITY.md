# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to our security team. All security vulnerabilities will be promptly addressed.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

Thank you for helping keep Synora Bridge secure.
